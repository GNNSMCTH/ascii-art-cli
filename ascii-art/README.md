# ASCII Art CLI

An interactive command-line tool that renders text as colorful ASCII art using [pyfiglet](https://github.com/pwaller/pyfiglet) and [colorama](https://github.com/tartley/colorama).

## Usage

```bash
pip install pyfiglet colorama
python app.py
```

Type any text, then choose to change the font or color. Press **Ctrl+C** to quit.

## Fonts

## Colors

Cyan, Magenta, Yellow, Green, Red, White, Blue.
